📚 ಪದಗಳ ಅರ್ಥ

ಪ್ರತಿ ಪದ ಟ್ಯಾಪ್ ಮಾಡಿ ಅರ್ಥ ನೋಡಿ!

ಹರಡು → ಹಾಸು / spread out
ಅನ್ನ → ಊಟ / cooked rice
ಬೇಳೆ → dal / ಬೇಳೆ ಕಾಳು
ಸಾರು → rasam / ತಿಳಿ ಸಾರು
ಪಲ್ಯ → ತರಕಾರಿ ಪದಾರ್ಥ
ದಂಟು → ಒಂದು ತರಕಾರಿ ಸೊಪ್ಪು
ಮುದಿರು → ಮುಚ್ಚು / fold up
ಬಾಳೆಲೆ → banana leaf / ಊಟದ ಎಲೆ
✏️ ಅ) ಸಂಖ್ಯೆ ಹೇಳಿ — ಒಂದರಿಂದ ಹತ್ತು
