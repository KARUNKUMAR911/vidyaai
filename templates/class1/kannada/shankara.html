ಶಂಕರಪ್ಪನ ಮನೆ

ಪಾಠ ೧೫ · ಗದ್ಯ
– ಸಮಿತಿ ರಚನೆ
ಹಂತ ೧ / ೬
🔒 ಮೊದಲಿನ ಹಂತ ಮುಗಿಸಿ!
📖 ಕಥೆ ಓದಿ
👇 ಕಥೆ ಓದಿ, ಅರ್ಥ ಮಾಡಿಕೊಳ್ಳಿ — ನಂತರ ಮುಂದೆ ಹೋಗಿ!
🌅 🏠
ಸಂಜೆಯ ಸಮಯ. ಶಂಕರಪ್ಪನ ಮನೆ. ಮನೆಯ ಬಾಗಿಲ ಬಳಿ ವೀಣಾ ನಿಂತಿದ್ದಳು. ಆಟವಾಡಲು ಅವಳು ಗೆಳೆಯರಿಗಾಗಿ ಕಾಯುತ್ತಿದ್ದಳು.
👴
ಮಹಾದೇವ ಅವರು ಮನೆಗೆ ಬಂದರು
ಮಹಾದೇವ ಅವರನ್ನು ನೋಡಿದ ವೀಣಾ ಸಂತಸಗೊಂಡಳು.
👧
ವೀಣಾ ಹೇಳಿದಳು
"ಬನ್ನಿ ಚಿಕ್ಕಪ್ಪ, ಬನ್ನಿ" — ಎಂದು ಮನೆಯೊಳಗೆ ಕರೆದುಕೊಂಡು ಹೋದಳು. ಕುಳಿತುಕೊಳ್ಳಲು ಹೇಳಿದಳು. ಕುಡಿಯಲು ನೀರು ತಂದುಕೊಟ್ಟಳು.
👧
ವೀಣಾ ಪ್ರೀತಿಯಿಂದ ಕೇಳಿದಳು
"ಚಿಕ್ಕಪ್ಪ ಚೆನ್ನಾಗಿದ್ದೀರಾ? ಚಿಕ್ಕಮ್ಮ ಮತ್ತು ತಮ್ಮನ ಆರೋಗ್ಯ ಹೇಗಿದೆ?"
👴
ಮಹಾದೇವ ಅವರಿಗೆ
ವೀಣಾಳ ಮಾತುಗಳನ್ನು ಕೇಳಿ ಚಿಕ್ಕಪ್ಪನಿಗೆ ಸಂತಸವಾಯಿತು.
ಮಹಾದೇವ ಅವರು ಅಣ್ಣ ಶಂಕರಪ್ಪನವರನ್ನು ಗೃಹಪ್ರವೇಶಕ್ಕೆ ಆಮಂತ್ರಿಸಿದರು. ಸ್ವಲ್ಪ ಹೊತ್ತು ಅಲ್ಲೇ ಇದ್ದು, ನಂತರ ತಮ್ಮ ಮನೆಗೆ ಹೊರಡಲು ಸಿದ್ಧರಾದರು.
👧
ವೀಣಾ ನಮಸ್ಕಾರ ಮಾಡಿ ಹೇಳಿದಳು
"ಇನ್ನೊಮ್ಮೆ ಬರುವಾಗ ಚಿಕ್ಕಮ್ಮನನ್ನೂ ತಮ್ಮನನ್ನೂ ಕರೆದುಕೊಂಡು ಬನ್ನಿ" — ಎಂದು ಹೇಳಲು ಮರೆಯಲಿಲ್ಲ.
📝 ಹೊಸ ಪದಗಳು
👆 ಪ್ರತಿಯೊಂದು ಪದವನ್ನು ಮುಟ್ಟಿ — ಅರ್ಥ ನೋಡಿ!
ಸಂಜೆ
👆 ಮುಟ್ಟಿ ನೋಡಿ
🌅 ಹಗಲು ಮುಗಿಯುವ ಸಮಯ (Evening)
ಚಿಕ್ಕಪ್ಪ
👆 ಮುಟ್ಟಿ ನೋಡಿ
👴 ಅಪ್ಪನ ತಮ್ಮ (Father's younger brother / Uncle)
ಸಂತಸ
👆 ಮುಟ್ಟಿ ನೋಡಿ
😊 ಸಂತೋಷ, ಖುಷಿ (Happiness / Joy)
ಪ್ರೀತಿ
👆 ಮುಟ್ಟಿ ನೋಡಿ
❤️ ಮಮತೆ, ಅಕ್ಕರೆ (Love / Affection)
ಆರೋಗ್ಯ
👆 ಮುಟ್ಟಿ ನೋಡಿ
💪 ಆರೋಗ್ಯಕರ ಸ್ಥಿತಿ (Health / Wellbeing)
ಗೃಹಪ್ರವೇಶ
👆 ಮುಟ್ಟಿ ನೋಡಿ
🏠 ಹೊಸ ಮನೆಗೆ ಮೊದಲ ಬಾರಿ ಪ್ರವೇಶ (Housewarming ceremony)
ನಮಸ್ಕಾರ
👆 ಮುಟ್ಟಿ ನೋಡಿ
🙏 ಗೌರವ ತೋರಿಸುವ ರೀತಿ (Greeting / Bow with respect)
ಆಮಂತ್ರಣ
👆 ಮುಟ್ಟಿ ನೋಡಿ
💌 ಕರೆ ಕೊಡುವುದು (Invitation)
ಮರೆ
👆 ಮುಟ್ಟಿ ನೋಡಿ
🧠 ನೆನಪು ಇಲ್ಲದಿರುವುದು (Forget)
ಸಿದ್ಧ
👆 ಮುಟ್ಟಿ ನೋಡಿ
✅ ತಯಾರಿ ಆಗಿರುವುದು (Ready / Prepared)
👥 ಕಥೆಯ ಪಾತ್ರಗಳು
📌 ಕಥೆಯಲ್ಲಿ ಯಾರು ಯಾರು ಇದ್ದಾರೆ — ನೋಡಿ!
👧
ವೀಣಾ
ಶಂಕರಪ್ಪನ ಮಗಳು. ತುಂಬಾ ಒಳ್ಳೆಯ ಹುಡುಗಿ. ಅತಿಥಿಗಳನ್ನು ಚೆನ್ನಾಗಿ ಉಪಚರಿಸುತ್ತಾಳೆ.
👴
ಮಹಾದೇವ (ಚಿಕ್ಕಪ್ಪ)
ಶಂಕರಪ್ಪನ ತಮ್ಮ. ವೀಣಾಳ ಚಿಕ್ಕಪ್ಪ. ಮನೆಗೆ ಭೇಟಿ ಕೊಡಲು ಬಂದಿದ್ದಾರೆ.
🧔
ಶಂಕರಪ್ಪ (ಅಪ್ಪ)
ಮನೆಯ ಯಜಮಾನ. ವೀಣಾಳ ತಂದೆ. ಮಹಾದೇವನ ಅಣ್ಣ.
ಕುಟುಂಬ ಸಂಬಂಧ:
ಶಂಕರಪ್ಪ 👨 ← ಅಣ್ಣ–ತಮ್ಮ → 👴 ಮಹಾದೇವ
ವೀಣಾ 👧 ← ಮಗಳು → 👨 ಶಂಕರಪ್ಪ
ವೀಣಾ 👧 ← ಚಿಕ್ಕ ತಂದೆ → 👴 ಮಹಾದೇವ
❓ ಪ್ರಶ್ನೆ–ಉತ್ತರ
🧠 ಕಥೆ ನೆನಪಿಸಿ — ಸರಿಯಾದ ಉತ್ತರ ಆರಿಸಿ!

೪
