Add Two Numbers!
Add two single digit numbers mentally!
💡 How to Add in Your Head!
Start from the bigger number, then count up!

5 fruits + 3 fruits = ?
Start at 5 → count 3 more → 6, 7, 8 ✅

Try: After 5 → 6, 7, 8 = 8!
Tap the correct answer!
🔢
Number Patterns!
Find the next number in the pattern!
💡 Spot the Pattern!
Numbers can go UP (increasing) or DOWN (decreasing)!

2, 4, 6, ? → numbers increase by 2 → 6+2 = 8 ✅
12, 9, 6, ? → numbers decrease by 3 → 6-3 = 3 ✅

Find the rule, then find the next number!
Tap the next number in the pattern!
🔲
Fill the Blanks!
Make both sides equal!
💡 Both Sides Must Be Equal!
The = sign means both sides have the same value!

6 + ? = 10 → 6 + 4 = 10 ✅
12 = ? + 8 → 12 = 4 + 8 ✅
3 = 0 + ? → 3 = 0 + 3 ✅
Tap the missing number!
📖
Word Sums!
Read the story and find the answer!
💡 Read, Think, Add!
Read the story carefully. Find the two numbers and add them!

5 books + 4 books = ?
Start at 5 → count 4 more → 9 books! ✅
Tap the correct answer!

True or False?
Is the statement correct?
💡 Check if it is Right!
Work out the sum mentally. Then decide:

✅ TRUE — if the answer is correct
❌ FALSE — if the answer is wrong

3 + 2 = 5 → TRUE ✅
6 + 6 = 4 → FALSE ❌ (6+6 = 12!)
Tap TRUE ✅ or FALSE ❌!
