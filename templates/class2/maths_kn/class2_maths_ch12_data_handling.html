What is Data?

📌 ದತ್ತಾಂಶ = ಮಾಹಿತಿ (Information)

ನಾವು ವಸ್ತುಗಳನ್ನು ಎಣಿಸಿ ಸಂಖ್ಯೆ ಬರೆದಾಗ — ಅದೇ ದತ್ತಾಂಶ!

When we count things and write numbers — that is Data!

ಎಣಿಸು → ಬರೆ → ಹೋಲಿಸು
Count → Write → Compare 📊

🎯 ಈ ಪಾಠದಲ್ಲಿ ಕಲಿಯುವುದು:

🐟 ಮೀನು ಎಣಿಸಿ ಕೋಷ್ಟಕ ತುಂಬುವುದು — Counting fish

🍎 ಹಣ್ಣು ಎಣಿಸಿ ದಾಖಲಿಸುವುದು — Recording fruit counts

🌸 ಹೂವುಗಳ ಚಾರ್ಟ್ — Flower chart

⚽ ಚೆಂಡುಗಳ ಹೋಲಿಕೆ — Comparing balls
