Yesterday • Today • Tomorrow
🐻
ಈ ದಿನ ಸೋಮವಾರವಾದರೆ,
ನಿನ್ನೆಯ ದಿನ → ಭಾನುವಾರ
ನಾಳೆಯ ದಿನ → ಮಂಗಳವಾರ

💡 ಮನಸ್ಸಿನಲ್ಲಿ ಇಡಿ

📌 ನಿನ್ನೆ = ಹಿಂದಿನ ದಿನ (Previous day)

📌 ಇಂದು = ಈ ದಿನ (Today)

📌 ನಾಳೆ = ನಂತರದ ದಿನ (Next day)

ಭಾನು → ಸೋಮ → ಮಂಗಳ → ಬುಧ → ಗುರು → ಶುಕ್ರ → ಶನಿ → ಭಾನು...
