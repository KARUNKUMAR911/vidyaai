What is Multiplication?

🌟 ಗುಣಾಕಾರ = ಪುನರಾವರ್ತಿತ ಸಂಕಲನ!

ಒಂದೇ ಸಂಖ್ಯೆಯನ್ನು ಮತ್ತೆ ಮತ್ತೆ ಕೂಡುವುದೇ ಗುಣಾಕಾರ.

Adding the same number again and again is called ಗುಣಾಕಾರ (Multiplication).

ಗುಣಾಕಾರದ ಚಿಹ್ನೆ: ×

"ಎಷ್ಟಪ್ಪ ಎಷ್ಟು? ನೀವು ಹೇಳಿದಷ್ಟು" ಆಟ!

Multiplication by Zero

🌟 ಸೊನ್ನೆಯಿಂದ ಗುಣಿಸಿದರೆ?

ಯಾವುದೇ ಸಂಖ್ಯೆಯನ್ನು ಸೊನ್ನೆಯಿಂದ ಗುಣಿಸಿದರೆ ಉತ್ತರ ಸೊನ್ನೆ!

Any number × 0 = 0 (Zero!)

ಯಾವುದೇ ಸಂಖ್ಯೆ × 0 = 0 🚫
ಯಾವುದೇ ಸಂಖ್ಯೆ × 1 = ಅದೇ ಸಂಖ್ಯೆ ✅
5 × 0 = 0
9 × 0 = 0
0 × 0 = 0
