Number Patterns!
Numbers follow rules — add or subtract to continue!

🌟 What is a Number Pattern?

A Number Pattern is an orderly arrangement of numbers!

👉 1, 2, 3, 4, 5… → Rule: Add 1 each time! ➕1️⃣

👉 2, 4, 6, 8, 10… → Rule: Add 2 each time! ➕2️⃣

👉 100, 90, 80, 70… → Rule: Subtract 10 each time! ➖🔟

👉 Every number follows a definite rule!

➕ Adding Pattern: +1
1→ 2→ 3→ 4→ 5→ 6
Rule: Add 1 each time!
➕ Adding Pattern: +2
2→ 4→ 6→ 8→ 10→ 12
Rule: Add 2 each time!
➖ Subtracting Pattern: −10
100→ 90→ 80→ 70→ 60
Rule: Subtract 10 each time!
📌 Number Pattern = Numbers in order following a rule!
➕ Add a number = pattern goes UP
➖ Subtract a number = pattern goes DOWN

Split Numbers & Creative Patterns!
Break numbers into parts & make your own patterns!

🌟 Splitting Numbers into Patterns!

We can split a number into smaller parts!

👉 2 = 1 + 1

👉 3 = 1 + 1 + 1

👉 4 = 1 + 2 + 1

👉 5 = 1 + 3 + 1

👉 See the pattern? The middle number goes up by 1 each time! 🔼

✂️ Splitting Pattern
2=1+1
3=1+1+1
4=1+2+1
5=1+3+1
6=1+4+1
7=1+5+1

🎨 Splitting 16 Many Ways!

One number can be split in different ways!

👉 16 = 8 + 8 (two 8s)

👉 16 = 4 + 4 + 4 + 4 (four 4s)

👉 16 = 2+2+2+2+2+2+2+2 (eight 2s)

👉 16 = 1+1+1+…+1 (sixteen 1s!)

🎯 Odd & Even Number Pattern

Numbers ending in 1, 3, 5, 7, 9 are ODD numbers! 🟡

Numbers ending in 2, 4, 6, 8, 0 are EVEN numbers! 🔵
